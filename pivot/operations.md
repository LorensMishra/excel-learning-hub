# Pivot Table Operations

## 1. Sorting in Pivot Table
- Used to arrange data in order
- Helps identify highest / lowest values

Types:
- Ascending
- Descending
- Sort by Values

Steps:
1. Right-click any value
2. Select **Sort**
3. Choose sorting option

---

## 2. Filtering in Pivot Table
- Used to display required data only
- Does not change source data

Types:
- Label Filters
- Value Filters
- Report Filters

Examples:
- Top 10 values
- Greater than / Less than
- Specific category selection

---

## 3. Grouping Data
- Combines data into logical ranges
- Reduces clutter

### 3.1 Date Grouping
- Year
- Quarter
- Month
- Day

### 3.2 Number Grouping
- Salary ranges
- Score bands

Steps:
1. Right-click field value
2. Select **Group**
3. Choose grouping option

---

## 4. Refreshing Pivot Table
- Required after source data update
- Ensures latest values are shown

Methods:
- Right-click → Refresh
- Pivot Analyze → Refresh
- Refresh All (multiple pivots)

---

## 5. Clearing & Rebuilding Pivot
- Remove fields to reset layout
- Does not delete source data
- Useful for redesigning reports

---

## 6. Common Operational Tips
- Always refresh before sharing report
- Avoid editing values manually
- Use grouping instead of formulas

---

## 7. Interview Tip
> Pivot operations include sorting, filtering, grouping, and refreshing to dynamically analyze data without modifying the source.
