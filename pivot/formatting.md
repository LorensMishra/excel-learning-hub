# Pivot Table Formatting

## 1. Pivot Table Styles
- Predefined styles available in Excel
- Improves readability and presentation

Steps:
1. Click inside Pivot Table
2. Go to **Design** tab
3. Choose a Pivot Table style

---

## 2. Report Layout Options
- Controls how fields are displayed

Types:
- Compact Form (default)
- Outline Form
- Tabular Form

Best practice:
- Use **Tabular Form** for reports and exports

---

## 3. Subtotals
- Shows summary at each group level
- Can be enabled or disabled

Options:
- Show at top of group
- Show at bottom of group
- Do not show

---

## 4. Grand Totals
- Shows overall total for rows and columns
- Can be turned ON or OFF

Steps:
Design → Grand Totals → On / Off

---

## 5. Number Formatting
- Formats numeric values (Currency, %, Decimal)
- Must be applied from **Value Field Settings**

Steps:
1. Value Field Settings
2. Number Format
3. Choose format

---

## 6. Conditional Formatting
- Highlights important values
- Works dynamically with Pivot Table

Examples:
- Top / Bottom values
- Color scales
- Data bars

---

## 7. Formatting Best Practices
- Avoid excessive colors
- Keep consistent number formats
- Do not format source data manually
- Refresh pivot after formatting changes

---

## 8. Interview Tip
> Pivot formatting improves readability using styles, layouts, totals, and number formats without changing data.
