# Pivot Table Structure & Components

## 1. Pivot Table Overview
- Pivot Table is built using **fields from source data**
- Analysis is done by dragging fields into specific areas
- Changes are dynamic and update instantly

---

## 2. Pivot Table Fields Area
Pivot Table has **four main sections**:

### 2.1 Rows
- Displays data vertically
- Used for categories (Product, Employee, Date)
- Shows unique values only

---

### 2.2 Columns
- Displays data horizontally
- Used for comparison across categories
- Best for limited unique values

---

### 2.3 Values
- Performs calculations on numeric data
- Default calculation: **Sum**
- Multiple value fields can be added

Common calculations:
- Sum
- Count
- Average
- Max / Min

---

### 2.4 Filters
- Filters the entire Pivot Table
- Used for high-level selection (Year, Region)
- Does not affect layout

---

## 3. Value Field Settings
- Controls how data is calculated and displayed

Steps:
1. Click dropdown on value field
2. Select **Value Field Settings**
3. Choose calculation type

Additional options:
- Show values as % of total
- Running total
- Difference from

---

## 4. Report Layout Options
- **Compact Form** (default)
- **Outline Form**
- **Tabular Form**

Tabular Form is preferred for:
- Readability
- Exporting data
- Using formulas outside pivot

---

## 5. Subtotals & Grand Totals
- Can be enabled or disabled
- Controlled from Pivot Table Design tab
- Helps in hierarchical analysis

---

## 6. Pivot Table Limitations
- Manual edits are not allowed inside pivot
- Source data structure should not change
- Large datasets may slow performance

---

## 7. Best Practices
- Rename fields clearly
- Keep rows concise
- Avoid too many columns
- Use filters or slicers for interactivity

---

## 8. Interview Tip
> Pivot Table structure consists of Rows, Columns, Values, and Filters, which together summarize and analyze data dynamically.
