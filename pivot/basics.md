# Basics of Pivot Table (Excel)

## 1. What is a Pivot Table?
- A Pivot Table is an Excel tool used to **summarize and analyze large datasets**
- Helps to generate reports without writing formulas
- Used for **comparison, totals, counts, averages, trends**

---

## 2. Why Use Pivot Tables?
- Fast data analysis
- Easy grouping and filtering
- Dynamic reports (auto-update on refresh)
- Widely used in Data Analyst / MIS / Business roles

---

## 3. Data Requirements
Before creating a Pivot Table, data must:
- Have **column headers**
- Contain **no blank rows or columns**
- Have **no merged cells**
- Store **one value per cell**
- Maintain same data type per column

✅ Best practice: Convert data to **Excel Table (Ctrl + T)**

---

## 4. Creating a Pivot Table (Steps)
1. Select any cell in the dataset
2. Go to **Insert → Pivot Table**
3. Confirm data range
4. Choose **New Worksheet / Existing Worksheet**
5. Click **OK**

---

## 5. Pivot Table Interface
- Pivot Table area (report output)
- Pivot Table Fields panel
- Drag-and-drop fields for analysis

---

## 6. Basic Pivot Table Areas
- **Rows** → Vertical categories
- **Columns** → Horizontal comparison
- **Values** → Calculations (Sum, Count, Avg)
- **Filters** → Filter entire report

---

## 7. Common Use Cases
- Sales summary by month / region
- Employee count by department
- Revenue vs target comparison
- MIS and management dashboards

---

## 8. Key Points to Remember
- Pivot Tables are **dynamic**
- Always **refresh** after data change
- Avoid manual editing inside pivot
- Rename fields for clarity

---

## 9. Interview Tip
> Pivot Table is used to convert raw data into summarized insights using rows, columns, values, and filters without complex formulas.
