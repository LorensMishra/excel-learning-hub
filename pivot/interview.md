# Pivot Table – Interview Notes

## 1. What is a Pivot Table?
- A Pivot Table is an Excel tool used to **summarize and analyze large datasets**
- It organizes data using **Rows, Columns, Values, and Filters**
- No complex formulas required

---

## 2. Why Are Pivot Tables Used?
- Fast data analysis
- Dynamic and flexible reports
- Easy comparison and summarization
- Commonly used in MIS, Data Analyst, and Business roles

---

## 3. Pivot Table Components
- Rows → Categories
- Columns → Comparisons
- Values → Calculations
- Filters → Report-level filtering

---

## 4. Common Interview Questions

### Q1. What are the main areas of a Pivot Table?
- Rows, Columns, Values, Filters

### Q2. What is the default calculation in Pivot Values?
- Sum

### Q3. How do you refresh a Pivot Table?
- Right-click → Refresh  
- Pivot Analyze → Refresh

### Q4. Difference between Calculated Field and Normal Formula?
- Calculated Field works on Pivot data
- Normal formulas work on worksheet cells

### Q5. What is the use of Slicers?
- Visual filtering and interactivity

---

## 5. Real-Time Use Cases
- Sales performance analysis
- Employee headcount reporting
- Financial summaries
- Management dashboards

---

## 6. Common Interview Traps
- Forgetting to refresh Pivot Table
- Numbers stored as text
- Editing Pivot values manually

---

## 7. Best Practices (Say This in Interview)
- Use clean and structured source data
- Convert data to Excel Table
- Use Tabular Form for reports
- Refresh before sharing

---

## 8. One-Line Interview Answer
> Pivot Table helps convert raw data into summarized insights using rows, columns, values, and filters dynamically.
