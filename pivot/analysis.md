# Pivot Table Analysis

## 1. Calculated Fields
- Used to create custom calculations inside Pivot Table
- Works on existing fields, not individual cells

Example:
- Profit = Sales − Cost

Steps:
1. Pivot Analyze → Fields, Items & Sets
2. Select **Calculated Field**
3. Enter name and formula

---

## 2. Show Values As
- Used to display values in different analytical formats
- Does not change actual data

Common options:
- % of Grand Total
- % of Row Total
- % of Column Total
- Difference From
- Running Total In

---

## 3. Percentage Calculations
- Helps in comparison and trend analysis
- Used for contribution analysis

Examples:
- Sales contribution by region
- Department-wise headcount %

---

## 4. Running Total
- Shows cumulative values
- Commonly used in time-based analysis

Example:
- Cumulative sales by month

Steps:
1. Value Field Settings
2. Show Values As → Running Total In
3. Select base field

---

## 5. Comparative Analysis
- Compare categories across time or regions
- Use Columns + Values combination
- Supports multi-level analysis

---

## 6. Limitations of Pivot Analysis
- Calculated fields have limited formula support
- Cannot reference individual pivot cells
- Complex logic may require helper columns

---

## 7. Best Practices for Analysis
- Keep calculations simple
- Use helper columns in source data if needed
- Rename calculated fields clearly

---

## 8. Interview Tip
> Pivot Table analysis uses calculated fields, percentages, and running totals to derive insights without altering source data.
