This repository is a centralized collection of Microsoft Excel learning resources designed for structured learning, quick reference, and long-term use. It covers essential and advanced Excel topics, including keyboard shortcuts, pivot table shortcuts, formulas, functions, and theory notes.

The goal of this repository is to help Excel users, data analysts, and professionals improve productivity, reduce dependency on the mouse, and develop a strong understanding of Excel concepts through well-organized notes and practical references.

## Repository Structure

- **excel_shortcut_keys/**  
  Complete list of Excel keyboard shortcuts using Ctrl, Shift, Alt, and all key combinations.

- **excel_pivot_shortcut_keys/**  
  Frequently used shortcut keys specific to Pivot Tables for faster data analysis.

- **excel_formula_notes/**  
  Notes and examples covering Excel formulas and functions used in real-world scenarios.

- **excel_theory_notes/**  
  Conceptual explanations of Excel features, tools, and best practices.

Each folder contains its own README file for focused documentation and easy navigation.

## Who This Repository Is For

- Excel beginners building strong fundamentals  
- Data analysts and professionals who want faster workflows  
- Anyone who prefers keyboard-driven productivity in Excel  

## How to Use

Browse the folders based on your learning needs. Use this repository as a quick reference, study guide, or continuous learning resource. The content is organized to allow easy expansion as new Excel topics are added.
